Editor

Editor

ResizeEditor
Resize image to given width and height
HumanCropEditor
Crop boundingbox region of human in image

Example

        
          from vimos.editor import ResizeEditor, HumanCropEditor
          from vimos.utils import Pipeline

          image = ...
          modifier = Pipeline([
                ResizeEditor(256, 256), 
                HumanCropEditor()
          ])
          
          image = modifier(image)
